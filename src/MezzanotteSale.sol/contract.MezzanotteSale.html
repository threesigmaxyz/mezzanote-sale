Structs

Sale

Defines the parameters for a sale period. This is created by the contract's Sale Admin. A sale is active when block.timestamp in interval [start, finish]. The limit per user of a whitelist sale is stored in the merkle root.

struct Sale {
    bytes32 root;
    bool whitelist;
    uint64 start;
    uint64 finish;
    uint64 price;
    uint8 limit;
    bool hasMaxMint;
    uint40 maxMint;
}
